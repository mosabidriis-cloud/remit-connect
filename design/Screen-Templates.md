# REOS Screen Templates

## Purpose

This document defines the standard layouts used across all REOS screens to ensure consistency, usability, and rapid development.

## General Layout Principles

- Consistent page header
- Responsive layout
- Primary information first
- Secondary information below
- Reusable sections
- Minimal scrolling where practical

---

## Dashboard Template

Sections:

- Page Header
- KPI Summary Cards
- Operational Alerts
- Main Activity Area
- Recent Activity
- Secondary Widgets

Purpose:
Provide a real-time operational overview and support decision-making.

---

## List Page Template

Sections:

- Page Header
- Summary Cards
- Search
- Filters
- Data Table
- Pagination

Purpose:
Efficient browsing, filtering, and selection of records.

---

## Detail Page Template

Sections:

- Header
- Summary Panel
- Tabs (when needed)
- Main Details
- Activity Timeline
- Related Information

Purpose:
Present complete information about a single entity.

---

## Transaction Workspace Template

Sections:

- Batch Summary
- Current Transaction
- Transaction Queue
- Processing Controls
- Progress Summary

Purpose:
Support sequential transaction processing with minimal distractions.

---

## Processing Screen Template

Sections:

- Progress Indicator
- Beneficiary Information
- Bank Information
- Account Number
- Amount
- Reference
- Confirmation Checklist
- Primary Action

Purpose:
Reduce operational errors and guide officers through each transaction safely.

---

## Form Page Template

Sections:

- Header
- Form
- Validation Messages
- Action Buttons

Purpose:
Provide a consistent experience for data entry.

---

## Modal Template

Sections:

- Title
- Description
- Content
- Primary Action
- Secondary Action

Purpose:
Handle confirmations and focused interactions.

---

## Empty State Template

Include:

- Title
- Explanation
- Suggested Action

---

## Error State Template

Include:

- Error Summary
- Possible Cause
- Recovery Action

---

## Loading State Template

Use:

- Skeleton loaders
- Progress indicators
- No layout shifting
